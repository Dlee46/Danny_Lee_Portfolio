Projects

Jeopardy

This was the first project created after learning HTML, CSS, and JavaScript. The application is a replica of the game Jeopardy. A game where categories are given with a set amount of questions. Each question is worth a certain amount of points.

Personal Profile

This was the second project created after learning RESTful routing, HBSJS, ExpressJS, NoSQL, MongoDB, and NodeJS. This application allows a user to store and create a profile that is curtailed to the User's needs.

Pokedex

This was the third project created after learning ReactJS along with useing NoSQL, MongoDB, ExpressJS, NodeJS (MERN STACK). The pokedex it self is pure css with information coming from an API. The application allows the user to create teams with different pokemon. It also allows the user to search general information about a pokemon.

Studio Ghibli Review

This was the fourth project created after learning Ruby and RAILS. The front-end was created with React with a SQL database using Postgresql. The application is a movie review app that allows users to leave reviews on Studio Ghibli Movies. The movie information comes from an API.
